1. What does auto do in grid rows?
Answer:auto in grid rows means the row height adjusts automatically based on its content.
2.Which row grows based on content?
Answer: The row with auto height grows based on its content.
3.Can rows have different heights?
Answer:Yes, rows can have different heights by using different values like px, auto, fr, or minmax().
4.What happens if content is very large?
Answer: If content is very large, an auto row expands, but a fixed-height row causes overflow.
5.Where are grid rows useful in real layouts?
Answer: Grid rows are useful for layouts like headers, content areas, footers, cards, and dashboards where vertical structure matters.
