1.What is the difference between a normal div and a grid container?

Answer: The difference between a normal div and grid container is that a grid container starts from the next line and ends at the new line , it is like a individual section that works as a block and grid is the special layout that helps to manage the sections and divide it into a column or row

2.Why does the layout look the same even after applying Grid?

Answer: The layout look same because , There is no command for the transformation of rows or columns

3.Are grid items created automatically?

Answer: No. Only the child elements become grid items. Grid does not create new boxes.

4.Can Grid work without rows or columns?

Answer: Grid can work , but there will be no visible changes without the rows or columns defined.

5.What do you think Grid needs to show visible changes?

Answer: Grid needs to use rows and columns to show visible changes.
